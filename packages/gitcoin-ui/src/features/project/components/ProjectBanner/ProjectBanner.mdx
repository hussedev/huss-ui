# ProjectBanner

The ProjectBanner component displays a banner image with an avatar, with options for positioning the avatar.

## Props

- **bannerImg**: string - CID for the banner image.
- **logoImg**: string - CID for the avatar image.
- **avatarPosition**: "center" | "left" | "right" (optional) - The position of the avatar on the banner. Defaults to "center".
